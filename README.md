## 주요 기능 (Features)
📸 하루 한 장의 감정 사진 업로드

🎭 5가지 감정 선택 (긍정, 좋음, 쇰쇰, 슬프름, 화남)

🎵 AI 기반 감정-맞춤 음악 추천

🗓️ 한 달 감정 카드 + 감정 리포트

🎼 이달의 감정 사운드트랙 자동 생성 및 공유

## 💡 왜 Melog인가?

📖 글로 쓰는 일기가 불편습니다.

📷 사지와 감정만으로 하루를 남겨보세요.

🎵 음악이 그냥의 기분을 대신 기억해줍니다.

## 모든 절차의 간소화, 입력은 간단하게 사진과 감정기반 음악 추천 AI기능
